SQL Rápido

Exemplos de comandos SQL para gerir a plataforma. Corre no Supabase SQL Editor.

Atualizar plano de utilizador
UPDATE profiles SET plan = 'pro' WHERE id = 'UUID_DO_UTILIZADOR';
Suspender utilizador
UPDATE profiles SET suspended = true WHERE id = 'UUID_DO_UTILIZADOR';
Dar permissão de admin
UPDATE profiles SET is_admin = true WHERE id = 'UUID_DO_UTILIZADOR';
Ver todos os profissionais e planos
SELECT name, email_from_auth, plan, suspended, created_at FROM profiles ORDER BY created_at DESC;
Apagar avaliação
DELETE FROM reviews WHERE id = 'UUID_DA_AVALIACAO';

💡 Para obter o UUID de um utilizador, clica em "Ver ID" na tabela de utilizadores acima.
